Pipe Assembly - An open source puzzle game

Use the W, A, S, and D keys to move around, and the <LEFT> and <RIGHT> arrow keys to rotate counter clockwise and clockwise, respectively. The game is still a work in progress, so a win state has not been implemented. The project is open source, and you can check out the code on github.com
